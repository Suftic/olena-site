# Olena Zolotariova — Website

Персональний сайт Олени Золотарьової: дитячий психолог (робота з особливими дітьми) + масаж для дітей і дорослих. Приймає у власному кабінеті вдома.

## Стек

Чистий HTML + CSS. Без збірки, без залежностей. Дизайн-токени на CSS-змінних, шрифти Fraunces + Inter з Google Fonts.

## Локальний запуск

```bash
python -m http.server 4321
# → http://localhost:4321
```

## Структура

```
index.html     — уся сторінка
styles.css     — токени й стилі секцій
assets/        — зображення (у планах)
```
